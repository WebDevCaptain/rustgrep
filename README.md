# Rustgrep

Rustgrep is a simple `grep`-like tool written in Rust.

## Usage

```bash
cargo run "search term" "file path"
```

## Example

```bash
cargo run "Staff" sample.txt
```

## Sample output

```txt
2: Staff Engineers provide technical leadership to one or more teams in an organization.
```
